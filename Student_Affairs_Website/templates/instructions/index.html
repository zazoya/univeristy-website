Instructions for users

How to add a student:

(1) Go to the navigation bar at the top of the page

(2) Select "Services".

(3) A drop down menu will appear,select "add student option".

(4) Fill the required information fields for the new student to be added such as his number , ID , Name..etc.

(5) Press register so that all the student's information will be saved.

(6) You must've already an account and logged-In.

How to Register department :

(1) Go to the navigation bar at the top of the page

(2) A drop down menu will appear,Select services.

(3) Select Register-Department.

(4) Enter student name/ID.

(5) Select department.

(6) You must've already an account and logged-In.

(7) Submit.

How to Edit/Delete student:

(1) Go to the navigation bar.

(2) A drop down menu will appear,select "services".

(3) Select "edit student".

(4) Search for active student.

(5) Edit student information or delete them.

(6) You must've already an account and logged-In.

How to view students:

(1) Go to the navigation bar.

(2) A drop down menu will appear,select "services".

(3) Select "Students".

(4) You can view active students by clicking "Select active" button , non-active ones by clicking"Select not active" or "view all" to view all students .

(6) You must've already an account and logged-In.

How to add announcement:

(1) Go to the navigation bar.

(2) A drop down menu will appear,select "services".

(3) Select "Add announcement".

(4) put down your announcement.

(5) You must've already an account and logged-In.

(6) Submit it.
